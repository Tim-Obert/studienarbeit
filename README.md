# Installation

## Vorraussetzungen
- Node 15
- yarn
- Python3.8

## Backend
- cd backend
- pip3 install -r requirements.txt
- cp data/db.json.example data/db.json
- mkdir data/videos

## Frontend
- cd frontend
- yarn install

# Ausführen
- cd backend && python3 main.py
- cd frontend && yarn serve

Öffne http://localhost:8080
